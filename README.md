# 洞穴之声 / The Call of A Cave

一个 2D 科幻洞穴虫群经营原型。玩家在荒芜洞穴中维持虫群的繁殖、劳作、医疗与防御；左侧营地逐步成为虫群城市，右侧是机械入侵者的战线。

完整规则、数值与分阶段任务见 [`GAMEPLAY_DESIGN.md`](GAMEPLAY_DESIGN.md)。

## 当前可玩循环

1. 开局拥有 300 金币、3 只幼虫和 2 枚虫卵；幼虫 120 金币，食物 50 金币。
2. 虫子以 360 秒经历幼体、成体、老年和死亡；健康持续下降，低健康与感染会加速衰老。
3. 自由成虫约每 120 秒产卵；派去工厂、兵营、医院或医学院后停止繁殖。
   出生区只是生成点，所有未派驻虫子都会在全图不受控制地乱跑；敌人出现后，主动安置虫群本身就是玩法。
4. 工厂产出可点击的 200 金币零件；兵营训练士兵；医院治疗感染和战伤；医学院培养清污博士。
5. 老虫与阵亡士兵留下污染。3 分钟后右侧出现机器人，需要部署士兵防守。
6. 最后一只虫死亡时进入 **虫群失联 / Colony Lost**。

## UI 与状态反馈

- **HUD**：显示金币、食物与幼体／成体数量，支持购买幼体、购买食物、暂停和切换语言。
- **虫体状态条**：所有虫子头顶显示健康与年龄；士兵额外显示战斗生命。
- **Toast**：顶部非阻塞通知，用于繁殖、建造、训练、感染、入侵和资源反馈。
- **Alert**：居中的确认／取消弹窗，当前用于退出确认，可复用于危险操作。
- **Block**：全屏黑幕，场景或关卡切换时渐入渐出。
- **Game Over**：虫群全灭后的双语失败界面。

## 视觉与场景

视觉关键词是 **深层洞穴、赭金荒地、青绿色生物荧光、废土科技营地**。当前为 16-bit 像素科幻风格。流沙与沙坑已从当前版本移除。

荒地、育巢、工厂、兵营、医院、医学院、进度条和全部 UI 都是 `Main.unity` 中可选中、移动、缩放和换图的真实对象。虫子、虫卵、敌人、博士、污染源和金属零件则通过各自 Prefab 实例化到 `WorldRoot`。

新增像素素材位于 `Assets/Resources/Art/ColonyV2`：老虫、机械敌人、变异油污染源和医学博士。工厂职员使用独立的 `bug_worker` 形象，普通成虫继续使用成虫行走表。

## 设施操作

- 点击未建成的工地进行建造。
- 按住 Shift 点击已建设施进行升级。
- 将自由成虫拖入工厂、兵营或医学院；将感染虫或受伤士兵拖入医院。
- 工厂职员永久派驻；医院成员需要玩家拖出；兵营完成训练后释放士兵。
- 医学院普通点击开始博士培养。每名博士固定消耗 300 金币和 5 食物。
- 点击虫子消耗 1 食物并恢复健康。

## 多语言约定（必须遵守）

项目以 `Assets/_Game/Data/LocalizationTable.asset` 为唯一文案表。任何面向玩家的新文字都必须：

1. 在 `LocalizationTable.cs` 的 `Keys` 中声明键名；
2. 在表中同时填写 `zh` 与 `en`；
3. UI 使用 `LocalizedText` 或 `LocalizationManager.Get(key)`；
4. 不在玩法脚本、预制体或场景中硬编码玩家可见文字。

## 运行

使用 Unity **2022.3.44f1** 或兼容的 2022 LTS 打开 `Assets/_Game/Scenes/Main.unity`。首次导入会自动烘焙场景设施、HUD 扩展、虫体状态条和动态实体 Prefab。若关闭了 Auto Refresh，请按一次 `⌘R`。

项目早期动画工作流参考 [`gorest-2d-animation-spritesheet-generator`](https://github.com/NO6KIKO/gorest-2d-animation-spritesheet-generator)；它不参与游戏运行时。
